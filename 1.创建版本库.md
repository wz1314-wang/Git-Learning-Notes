**配置git**
```bash
git config --global user.name "Darren"
git config --global user.email "xxxx@126.com"
```

**创建名为learngit的仓库**
```bash
mkdir learngit
cd learngit
pwd
git init
```

**在仓库中编写、暂存并提交文件**
- 先git add 把文件从「工作区」暂存到「暂存区」，再git commit把暂存区的内容提交到仓库，
	- **git add**：**把工作区的当前状态同步到暂存区**
	- **git commit：把暂存区内容提交到版本库**
	- **暂存区存储是 “要提交的修改（变更）操作，而非文件本身”**
- 用vi编辑器创建or打开readme.txt文件后：
	- 输入  i  进入编辑模式（Insert）
	- Git is a version control system. Git is free software.
	- 按 esc 回到命令模式
	- 命令模式下输入 :wq  保存文件并退出 vi
```bash
vi readme.txt
git add readme.txt
git commit -m "wrote a readme file"
```
当然也可以一次add多个文件，最后一次提交
```bash
git add file1.txt file2.txt file3.txt
git commit -m "add 3 files."
```

**查看文件内容**
```bash
cat readme.txt
cat readme.txt test.txt
```
